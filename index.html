Git & GitHub Beginner Guide

1. What is Git?

Git is a Version Control System that helps track code changes, collaborate with developers, and restore previous versions.

2. What is GitHub?

GitHub is a cloud platform that hosts Git repositories online.

3. Initialize Git Repository
git init

Creates a new Git repository.

4. Check Status
git status

Shows tracked and untracked files.

5. Add Files
git add .

Add all files to staging area.

6. Commit Changes
git commit -m "Initial Commit"

Saves your project snapshot.

7. Push to GitHub
git push origin main

Uploads code to GitHub.

8. Clone Repository
git clone https://github.com/user/repo.git

Downloads repository from GitHub.

9. Pull Latest Changes
git pull

Gets latest updates from GitHub.
